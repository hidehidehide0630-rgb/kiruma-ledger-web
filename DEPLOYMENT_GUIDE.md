# 🚀 Vercel デプロイ・最終手順ガイド

GitHubへのプッシュ、お疲れ様でした！残りの作業はブラウザ上での数クリックのみです。
帰宅後、以下の手順でアプリを世界（自分専用）に公開しましょう。

---

## 1. Vercel でプロジェクトをインポート
1. [Vercel New Project](https://vercel.com/new) にアクセスします。
2. GitHubリポジトリ一覧から **`kiruma-ledger-web`** を探し、**[Import]** ボタンをクリックします。

## 2. 環境変数（Environment Variables）の設定 ※最重要
デプロイ設定画面の「Environment Variables」セクションで、以下の2つを必ず追加してください。

| 名前 (Name) | 値 (Value) | 確認方法 |
| :--- | :--- | :--- |
| **`DATABASE_URL`** | `postgresql://...` | 同フォルダの `.env` ファイルからコピー |
| **`GEMINI_API_KEY`** | `AIzaSy...` | 同フォルダの `.env` ファイルからコピー |

**設定方法:**
- `Key` に名前、`Value` に値を貼り付けて **[Add]** を押せばOKです。

## 3. デプロイの実行
- 一番下の **[Deploy]** ボタンを押します。
- 1〜2分待つと「Congratulations!」と表示され、アプリのURLが発行されます。

## 4. スマホからのアクセス
1. 発行されたURL（`https://xxx.vercel.app`）をスマホに送ります。
2. スマホのブラウザで開き、**「ホーム画面に追加」** しておくと、普通のアプリと同じ感覚でサクッと起動できるようになります。
